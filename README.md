# DVF - Streamlit

**This is a project to visualize the data from the French government's [DVF](https://www.data.gouv.fr/fr/datasets/demandes-de-valeurs-foncieres/) (Demandes de Valeurs FonciÃ¨res) dataset.** 

> The dataset contains information about the sales of real estate in France. 
> We use in this project data from years 2019 and 2020.

## Installation

```bash
pip install streamlit
streamlit run main.py
```
